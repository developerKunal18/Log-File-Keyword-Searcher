# Log File Keyword Searcher

A Python script that searches for a keyword in a log file.

## Description
This project scans a log file and prints all lines containing a specified keyword.
It also counts the total number of matches.

## Features
- Keyword search
- Line-by-line scanning
- Match counting
- Useful for debugging logs

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Place a log file in the project folder
3. Run:

   python log_search.py

4. Enter file name and keyword

## Example

Input:
ERROR

Output:
ERROR Connection failed  
ERROR Timeout occurred  

Total matches: 2
